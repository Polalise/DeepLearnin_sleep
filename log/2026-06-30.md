- Accepts coarse presets:
  - overall activity increased / similar / decreased / unknown

- `data/processed/pre_sleep_forecasting/prototype_outputs/quick_preset_feature_source_summary.csv`

Interpretation:

- This is not a label-entry tool.
- It does not ask the user for the correct answer.
- It is a limited-input prototype mode that fills the model's expected feature contract and lets the existing trained MLP produce the prediction.
- Missing or unsupported features are left missing and handled by the fitted training-time imputer.
